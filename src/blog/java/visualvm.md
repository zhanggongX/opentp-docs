### 本地分析
在启动应用程序后，将打开 VisualVM 的主窗口。缺省情况下，“应用程序” 窗口显示在主窗口的左窗格中。在 “应用程序” 窗口中，可以快速查看本地和远程 JVM 上运行的 Java 应用程序。
local 就是本地运行的 Java 应用。
> 生产环境用处不大，可以用来学习使用。
